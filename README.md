# Airline Ticket Price Analysis ✈️

## Project Overview
This project analyzes an airline ticket dataset from Kaggle to identify pricing trends, airline efficiency, and the best "value for money" options based on distance traveled per USD.

## Key Insights
* **Most Frequent Airlines:** Flynas, Emirates, and Saudia.
* **Best Value:** Flynas offers the best distance-to-cost ratio (~4.19 km per USD).
* **Data Cleaning:** Handled column standardization and verified data integrity (zero nulls/duplicates).

## Tech Stack
* **Python** (Pandas for data manipulation)
* **Jupyter Notebook**
* **Matplotlib/Seaborn** (for visualization)

## How to Run
1. install pandas
2. import the data into jupyter notebook.
3. do the analysis(EDA, Insights)
